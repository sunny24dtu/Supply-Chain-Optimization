 Supply Chain Optimization using Linear Programming

  Overview
This project focuses on solving a real-world supply chain problem using Linear Programming. The objective is to minimize transportation cost while efficiently allocating resources between warehouses and customers.

  Objective
- Minimize total transportation cost  
- Satisfy supply constraints from warehouses  
- Meet demand requirements of customers  

  Approach
- Modeled the problem as a Linear Programming optimization task  
- Defined decision variables for shipment quantities  
- Applied constraints for supply and demand balance  
- Solved using PuLP in Python  

 Tools & Technologies
- Python  
- PuLP (Linear Programming Library)  

 Results
- Achieved optimal shipment allocation  
- Minimized total transportation cost  
- Improved efficiency in resource distribution  

 Key Learnings
- Understanding of Linear Programming concepts  
- Practical implementation of optimization problems  
- Importance of constraint-based decision making
